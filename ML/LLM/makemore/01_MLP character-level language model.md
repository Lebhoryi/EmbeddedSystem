## 构建数据集

进行训练前要先构建数据集，train dev 和 test 数据集分别占总数据量的 80/10/10，目前我们总共有 32033 个名字可以使用。

- 训练集：用于模型的学习过程，模型通过训练集中的数据不断调整参数（例如神经网络的权重），最小化 loss
- 验证集：用于模型的调优过程，在训练过程中，模型并不会用验证集来更新参数，而是用来评估模型在未见过的数据上的表现。
  - 用于超参数调整，例如学习率、网络层数、正则化系数
  - 用于早停，防止过拟合：如果模型在训练集上的表现越来越好，但是在验证集上表现变差，说明模型开始记忆训练数据而不是泛化规律
- 测试集：在模型的训练和调优都完成后，才使用测试集来评估模型的泛化能力（在未见过的数据上的表现）
  - 用于模型的最终评估，测试集在整个开发过程中都不能被用来训练或调优模型

###  构建方法

模型训练的目标是根据前 3 个字符（上下文长度可以修改）来预测下一个字符，因此我们将名字列表转化为神经网络训练用的输入输出对。

遍历所有名字，构建数据集，示例如下：

```

.me ---> l

..a ---> l

onn ---> i
nni ---> .

... ---> m
..m ---> o
```

`...` 代表开始，`.` 代表结束，可以看到一个由 5 个字符组成的单词，如果上下文长度为 3 的话，可以生成 6 组数据用于训练。最终生成的数据数量如下：

## 构建神经网络

### 模型结构

我们准备构建一个前馈神经网络，具体为带有嵌入层的单隐藏层 MLP。

- 嵌入层：C (27x10) 将输入索引映射到 10 维向量，不涉及激活函数，常视为预处理层。

- 隐藏层：W1 (30x200) + b1 (200) + tanh 激活函数，这是唯一的隐藏层（非线性变换）。

- 输出层：W2 (200x27) + b2 (27)，无激活（直接输出 logits），后接 `softmax` 或 `cross-entropy`

上下文：输入上下文长度为 3（block_size=3），嵌入后展平为 30 维（3x10），然后进入隐藏层。这是一个浅层网络，适合简单任务如名字生成。

“该模型是一个前馈神经网络 (Feedforward Neural Network)，具体为带有嵌入层的单隐藏层 MLP。输入为上下文序列（长度 3 的字符索引），通过嵌入矩阵（27x10）映射为低维表示；随后展平后输入隐藏层（线性变换 30→200，tanh 非线性激活）；输出层（线性变换 200→27）产生下一个字符的 logits。优化使用交叉熵损失和梯度下降，适用于生成式任务如名字序列预测。”

# 基础知识

## 什么是前馈神经网络？

前馈神经网络（Feedforward Neural Network, FFNN）是神经网络的最基本类型之一，常用于分类、回归等任务。

#### 1. **定义**
- 前馈神经网络是一种信息单向流动的神经网络模型，其中数据从输入层经过隐藏层直接流动到输出层，没有反馈循环（即没有从输出返回到输入的路径）。
- 它也被称为“多层感知机”（Multi-Layer Perceptron, MLP）当它有多个隐藏层时。核心思想是模拟大脑的神经元连接，通过层级结构学习输入到输出的映射。

#### 2. **结构**
- **输入层 (Input Layer)**：接收原始数据（如向量、图像像素）。每个节点代表一个输入特征。
- **隐藏层 (Hidden Layers)**：一或多个中间层，每个层包含多个神经元（节点）。每个神经元通过权重矩阵和偏置进行线性变换，然后应用激活函数（如 sigmoid、ReLU 或 tanh）引入非线性。
- **输出层 (Output Layer)**：产生最终结果（如分类概率、回归值）。通常使用 softmax（分类）或线性激活（回归）。
- **连接方式**：全连接（Fully Connected），即每层的所有神经元都与下一层的所有神经元相连。信息流动是单向的：输入 → 隐藏 → 输出，没有循环。

示例结构（简单 2 层 FFNN）：
```
输入层 (e.g., 3 维) → 隐藏层 (e.g., 200 神经元, tanh) → 输出层 (e.g., 27 维)
```

#### 3. **工作原理**
- **前向传播 (Forward Propagation)**：数据从输入层逐层向前传递。
  - 对于每个层：输出 = 激活函数(权重矩阵 × 输入 + 偏置)。
  - 示例数学表示：隐藏层 h = tanh(W1 * x + b1)，输出 o = W2 * h + b2。
- **训练过程**：使用反向传播 (Backpropagation) 和梯度下降优化参数（权重和偏置），最小化损失函数（如交叉熵）。
- **无循环**：不像循环神经网络 (RNN)，它不处理时间序列或反馈；每一步计算都是独立的。

#### 4. **特点和优缺点**
- **优点**：
  - 简单易实现，计算效率高。
  - 适合静态输入（如图像分类、数值预测）。
  - 通用性强，能逼近任何连续函数（通用逼近定理）。
- **缺点**：
  - 无法捕捉序列或时间依赖（e.g., 语言中的上下文），因为没有记忆机制。
  - 对于复杂任务，可能需要很深的层，导致梯度消失/爆炸问题。
- **与其它网络的区别**：
  - **对比 RNN**：RNN 有循环，能处理序列数据（如文本），但可能有梯度问题。
  - **对比 CNN**：CNN 专为网格数据（如图像）设计，使用卷积层共享权重。
  - **对比 Transformer**：Transformer 使用自注意力机制，无循环但能并行处理序列，比 FFNN 更先进。
